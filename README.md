Todo List App

A full-stack Todo List application built with React, MobX, Node.js, Express, Sequelize, PostgreSQL, JWT, and bcrypt.

Features
	•	User authentication with JWT and bcrypt
	•	Create, edit, delete, and view tasks
	•	Tasks can have date, time, and status
	•	Filter tasks by all, completed, or upcoming
	•	Responsive frontend with React and MobX
	•	Error and success messages using global state

Tech Stack

Frontend: React, MobX, React Router, Axios
Backend: Node.js, Express, Sequelize, PostgreSQL
Authentication: JWT, bcrypt
